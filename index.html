Introdução Como Usar Grid Componentes

Introdução

Mirana é uma ferramenta que tem por objetivo facilitar o desenvolvimento de layouts responsivos, no menor espaço de tempo possível, o que proporciona ao desenvolvedor uma possibilidade de aumentar sua produtividade, o diferencial do framework é sua simplicidade, possui classes bem intuitívas e é bem leve.

Leve

Arquivo minificado com menos de 4kb

100% responsivo

Sistema de grid baseado no flexbox layout e inspirado no bootstrap

Intuitívo

Classe fáceis de decorar e com nomenclatura bem intuitíva

Como usar

Faça o download, adicione a pasta CSS na raiz do seu projeto, por último coloque o código no head abaixo do seu html

<!--HTML-->
<link rel="stylesheet" href="css/normalize.css">
<link rel="stylesheet" href="css/mirana.min.css">
